rb-cli setrsrc

Write the resource fork of an existing HFS / HFS+ file from a host file

Usage

Usage: setrsrc --from-file <FROM_FILE> <IMAGE> <PATH>

Arguments

<IMAGE>
Image reference (`path` or `path@N`)
<PATH>
Absolute Mac path of the file whose resource fork should be replaced

Options

--from-file
Host file whose contents become the new resource fork
