rb-cli shrink

Re-encode a disk image into a CHD with trailing zero padding dropped (SGI/IRIX today)

Usage

Usage: shrink <INPUT> <OUTPUT>

Arguments

<INPUT>
Source image (raw `.img` or `.chd`). Must contain an SGI volume header at sector 0
<OUTPUT>
Destination CHD path. Must end in `.chd`, must not already exist, and must not resolve to the same file as `input`
