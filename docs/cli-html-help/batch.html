rb-cli batch

Apply a JSON-described sequence of FS operations to an image as one transaction-like batch

Usage

Usage: batch [OPTIONS] <SCRIPT>

Arguments

<SCRIPT>
Path to the batch JSON script

Options

--target
Override the script's `target` field (`path` or `path@N`)
--dry-run
Validate + print the plan, don't apply
--continue-on-error
Continue with remaining ops after a non-fatal failure. Default is stop-on-first-error
