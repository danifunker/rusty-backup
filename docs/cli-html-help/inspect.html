rb-cli inspect

Whole-disk aggregate read-only view (partition table + per-partition summary + CHD metadata when applicable)

Usage

Usage: inspect [OPTIONS] <IMAGE>

Arguments

<IMAGE>
Image path. `inspect` always reads the whole disk — there is no `@N` form. For per-partition detail use `show fs-info IMG@N`
