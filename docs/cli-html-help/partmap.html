rb-cli partmap

Edit the partition table (add / resize / delete / set-type / set-bootable). Partition *data* is never moved

Usage

Usage: partmap <COMMAND>

Subcommands

  • addAdd a new partition entry
  • resizeResize an existing partition entry (changes size only — data is not moved)
  • moveMove a partition entry to a new start LBA (does not move data)
  • deleteDelete a partition entry (zeroes the slot)
  • set-typeChange a partition's type byte / GUID / APM type string
  • set-bootableToggle the bootable flag (MBR active-partition bit; RDB flag)
  • applyApply a JSON script of edits as one transaction
