rb-cli command-line help

Headless image-construction CLI for rusty-backup

Global options

Options

--log-level
Diagnostic verbosity for stderr logs. Falls back to `[defaults] log-level` from the config; built-in default `warn`
-q / --quiet
Suppress all stderr output except errors and the final result. Mutually exclusive with `--log-level debug|trace`
--progress
Progress bar behavior. Built-in default `auto`; `never` is the safest setting inside CI / cron / wrapper scripts
--color
ANSI color usage. Honors the `NO_COLOR` env var when set. Built-in default `auto`
--log-file
Mirror full trace-level log output to PATH regardless of `--log-level`. Useful on Windows cmd where redirection is awkward
--config
Path to a config file. Overrides the platform default location. See `rb-cli config path` for what that location is

  • expandExpand a classic-HFS volume to a new size + allocation block size by cloning into a fresh APM disk image (default) or a bare HFS image (--to-hfv). Accepts APM-wrapped sources or raw single-partition HFS images.
  • fsckCheck (and optionally repair) a filesystem
  • getExtract a file from a filesystem to the host
  • growGrow a disk image by `--add SIZE` of trailing zero-padding so a subsequent `partmap` edit can place a new partition
  • inspectWhole-disk aggregate read-only view (partition table + per-partition summary + CHD metadata when applicable)
  • install-completionsInstall shell completions to the user-scoped canonical location
  • lsList a directory inside a filesystem
  • mkdirCreate a directory inside a filesystem
  • newCreate a blank single-partition image (superfloppy or, in Phase D, partition-table-wrapped)
  • opticalOptical-media verbs (rip / convert / browse / extract)
  • partmapEdit the partition table (add / resize / delete / set-type / set-bootable). Partition *data* is never moved
  • putCopy a host file (or zero-fill / write boot blocks) into a filesystem
  • resizeResize the filesystem at IMG@N to a new size (FAT/NTFS/exFAT/HFS+/ ext/btrfs/SFS/PFS3/AFFS/EFS — whichever magic matches)
  • restoreRestore a backup folder to a target image or device
  • rmDelete a file or directory from a filesystem
  • setrsrcWrite the resource fork of an existing HFS / HFS+ file from a host file
  • showFocused read-only queries
  • shrinkRe-encode a disk image into a CHD with trailing zero padding dropped (SGI/IRIX today)
  • terminalOpen an interactive rb-cli shell (rustyline-based REPL)
  • writeStream an image file onto a block device
