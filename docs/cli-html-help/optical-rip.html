rb-cli optical rip

Rip a physical CD/DVD drive to a disk image file

Usage

Usage: rip [OPTIONS] --device <DEVICE> --output <OUTPUT>

Options

--device
Source drive (e.g. `/dev/sr0`, `disk6`, `\\.\E:`). See `rb-cli show devices`
--output
Output path: `.iso` for `--format iso`, `.cue` for `--format bincue`
--format
--eject
Eject the disc after a successful rip
