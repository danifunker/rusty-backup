rb-cli fsck

Check (and optionally repair) a filesystem

Usage

Usage: fsck [OPTIONS] <IMAGE>

Arguments

<IMAGE>
Image reference (`path` or `path@N` for the 1-based partition index)

Options

--checkonly
Scan only. Never prompt, never repair. Exits non-zero on issues
--repair
Auto-repair detected issues without prompting
--prompt-timeout
Seconds to wait for an interactive repair confirmation before resolving to "No" (default 30; or `[fsck] prompt-timeout` from the config file when set). `0` waits indefinitely (TTY only)
