rb-cli write

Stream an image file onto a block device

Usage

Usage: write [OPTIONS] <IMAGE> <DEVICE>

Arguments

<IMAGE>
Source image file
<DEVICE>
Destination block-device path: - Linux: `/dev/sdX` or `/dev/nvmeXnY` - macOS: `/dev/diskN` / `/dev/rdiskN` - Windows: `"\\.\PhysicalDriveN"` (quote for PowerShell)

Options

--yes
Required confirmation. Skips the prompt but never the safety summary printed on stderr
--write-to-system-disk
Allow writing to the system boot disk (refused by default)
