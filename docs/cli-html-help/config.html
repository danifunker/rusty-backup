rb-cli config

Manage the rbcli.conf config file

Usage

Usage: config <COMMAND>

Subcommands

  • initWrite a commented template to the user's config location (or to `--path PATH`). Refuses to overwrite an existing file unless `--force`
  • pathPrint the resolved config-file path (whether it exists or not)
  • showPrint the loaded config as `section.key = value` lines
