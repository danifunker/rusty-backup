rb-cli install-completions

Install shell completions to the user-scoped canonical location

Usage

Usage: install-completions [OPTIONS]

Options

--shell
Force the shell instead of auto-detecting from `$SHELL` / `$PSModulePath`
--prefix
Override the install prefix (rarely needed). When set, the file is written under `PREFIX/<canonical-subdir>`
--print
Print the script to stdout instead of writing to disk
--uninstall
Remove the installed completion file. No-op if it doesn't exist
