rb-cli grow

Grow a disk image by `--add SIZE` of trailing zero-padding so a subsequent `partmap` edit can place a new partition

Usage

Usage: grow --add <ADD> <IMAGE>

Arguments

<IMAGE>
Image to grow

Options

--add
Bytes of zero-padding to add at the end (e.g. `512M`, `2G`)
